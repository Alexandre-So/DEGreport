## Installation

This is an [R][] package.

### [Bioconductor][] method

```r
source("https://bioconductor.org/biocLite.R")
biocLite("DEGreport")
```

### [devtools][] method

```r
install.packages("devtools")
devtools::install_github("lpantano/DEGreport")
```

Load the package and learn what to do:

`browseVignettes("DEGreport")`
